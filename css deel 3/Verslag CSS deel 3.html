Opdracht 2

Zijn alle properties die de generator voorziet werkelijk nodig? Probeer dit uit.

Neen, bepaalde properties zijn eigenlijk gewoon de default settings van een tabel die automatisch zijn toegekend door de tabelgenerator. De generator gebruikt voor bepaalde properties een browser prefix. Zijn die voor jouw browser (nog steeds) nodig? Voor bepaalde properties wel, maar voor anderen niet. Bijvoorbeeld bij Border radius hoeft dit niet meer, maar bij gradient moet dit wel nog steeds.

Opdracht 3

De generator geeft de elementen van elke even rij een 'alt' class: ... De generator voorziet de cellen in deze rijen van een andere achtergrondkleur, zoek op waar dit precies staat in de gegenereerde CSS.

Dit staat onder .datagrid table tbody .alt td{ background: #000000; color: #E010BE}. Waarom heeft de volgende CSS regel: .onbekend {background-color:red;} geen effect als je deze toepast op een cel in een rij met class="alt", maar wel in een rij zonder deze 'alt' class? Hint: het heeft te maken met de specificiteit van de regels. De regel waarin class=”alt” staat bevat 2 Klassen wat al automatisch de .onbekend class overheerst.
